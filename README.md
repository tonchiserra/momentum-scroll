# Momentum Scroll

Function based on [BCJdevelopment/butter.js](https://github.com/BCJdevelopment/butter.js).

## How to use
Include a script link into your HTML file:
```
<script src="momentum-scroll.js"></script>
```

Add id="scroll" into body tag child
```
<body>
    <main id="scroll"></main>
</body>
```

This is all.
